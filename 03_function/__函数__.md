#### 常用的数学函数

| 函数      | 功能描述                            |
| :-------- | :---------------------------------- |
| abs(x)    | 返回数值的绝对值                    |
| ceil(x)   | 返回数值的上入整数                  |
| exp(x)    | 返回 e 的 x 次幂                    |
| fabs(x)   | 返回浮点数的绝对值                  |
| floor(x)  | 返回浮点数的下舍整数                |
| log(x)    | 返回 x 的自然对数                   |
| log10(x)  | 返回以 10 为基数的 x 的对数         |
| max(x, y) | 返回给定参数的最大值                |
| min(x, y) | 返回给定参数的最小值                |
| pow(x, y) | 返回以 x 为底数、y 为指数的运算结果 |
| round(x)  | 返回浮点数 x 的四舍五入值           |
| sqrt(x)   | 返回数值 x 的平方根                 |

#### 常用的三角函数

| 函数        | 功能描述                          |
| :---------- | :-------------------------------- |
| acos(x)     | 返回 x 的反余弦弧度值             |
| asin(x)     | 返回 x 的反正弦弧度值             |
| atan(x)     | 返回 x 的反正切弧度值             |
| cos(x)      | 返回 x 的弧度的余弦值             |
| hypot(x, y) | 返回欧几里德范数 sqrt (x*x + y*y) |
| sin(x)      | 返回的 x 弧度的正弦值             |
| tan(x)      | 返回 x 弧度的正切值               |
| degrees(x)  | 将弧度转换为角度                  |
| radians(x)  | 将角度转换为弧度                  |
